
# yarn-aliases

  Commonly used yarn aliases. Fork for more!

## Installation

```bash
$ source aliases.sh
```

## Aliases

| Alias | Command |
|-------|---------|
| `yi`  | `yarn install` |
| `ya`  | `yarn add` |
| `yad`  | `yarn add --dev` |
| `yga`  | `yarn global add` |
| `yr`  | `yarn run` |
| `yt`  | `yarn test` |
| `yit`  | `yarn install && yarn test` |
| `yl`  | `yarn link` |
| `yul` | `yarn unlink` |
